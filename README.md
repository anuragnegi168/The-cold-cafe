# ☕❄️ The Cold Cafe

A Python-based Cafe Management System built using modular programming and OOP concepts.

* 📋 Menu system
* 🧾 Bill generation
* 💰 Tax calculation
* 🎯 Discount system
* 🧩 Modular code structure
* 🔄 Easy to extend
